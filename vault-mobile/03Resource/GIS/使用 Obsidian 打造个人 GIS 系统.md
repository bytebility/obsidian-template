# 使用 Obsidian 打造个人 GIS 系统

## 使用 Obsidian 打造个人 GIS 系统

### 需求
1. 徒步轨迹管理
	1. 手表徒步轨迹导入
	2. 二步路的徒步轨迹导入
2. 兴趣点管理
	1. 好玩的：景点、营地、公园、商场
	2. 好吃的：餐馆、采摘果园、集市
3. 个人经历位置管理
	1. 居住位置
	2. 工作位置
	3. 朋友位置

## MCP 示例

```bash
使用高德地图MCP 搜索杭州各前10 的好吃的和好玩的地点。  
  
1. 地点写入到笔记中，使用[地点名称](geo:wgs84维度,wgs84经度)，并添加简要介绍  
  
2. 需要调用`坐标转换工具.py`里面的`gcj02towgs84`方法将返回的坐标转换为wgs84​​
```
